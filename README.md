# Controle de Estoque - Java POO

Projeto desenvolvido para fins de estudo de Programação Orientada a Objetos (POO) em Java. O sistema permite o gerenciamento básico de produtos em um estoque.

## 🚀 Funcionalidades
* Cadastro de produto com Nome, Preço e Quantidade.
* Adição de produtos ao estoque existente.
* Remoção de produtos do estoque.
* Cálculo automático do valor total em estoque.
* Exibição formatada dos dados do produto (toString).

## 🛠️ Conceitos Aplicados
* **Classes e Objetos**: Estrutura baseada na entidade `Product`.
* **Construtores**: Garantia de que o objeto seja instanciado com dados válidos.
* **Métodos de Instância**: Lógica de negócio encapsulada dentro da classe de entidade.
* **Organização em Pacotes**: Separação clara entre `application` e `entities`.

## 💻 Como executar
1. Clone o repositório.
2. Certifique-se de ter o JDK instalado.
3. Execute a classe `Program.java` localizada no pacote `application`.
